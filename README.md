# riemann-zeta-sigmoid-transition
这是一个关于黎曼ζ函数最近邻间距统计的S型过渡的演示代码。（riemann-zeta-sigmoid-transition）

注意事项：
①200万数据集太大上传不了，需要自己去官网获取，然后放在相同文件夹下即可，记得把200万数据集的文件名改为zeros200.txt。（数据可从 https://www-users.cse.umn.edu/~odlyzko/zeta_tables/ 获取。）

②运行100K和2M方差图.py，得到图一与图二。
③运行CUE结果画图.PY,得到图三。
④运行CUE计算.py得到与CUE有关的数据。
⑤运行L函数画图.PY得到图四与图五。（使用1-5-5.3-r1-0-0.zeros.txt得到图四，使用1-2800-2800.1621-r0-0-0.zeros.txt得到图五）
